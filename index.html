As a project  assistant and IT support lead I:  Provide IT support for all eLimu project which involves determining project  devices, installing the software, training the teachers and providing after  sales support  Organize focus groups to get insight on how to improve the products  Work with the Project Manager to successfully implement all projects online Marketing and pitching to new clients.
